e) Mode opératoire du robot

Le robot dispose de 3 modes opératoires :

  • Mode Manuel – Calibration du robot : le robot définit automatiquement les positions Zéro de chaque axe
  • Mode Collaboratif – Programmation : la programmation, l'édition, les chargements et l'exécution des programmes
  • Mode Manuel – Free Motion : les axes du robot peuvent être déplacés à la main.

Le mode actuel peut être déterminé via la couleur de l'anneau lumineux.

Mode Manuel - Calibration du robot : la phase de calibration du robot est entièrement automatique et obligatoire afin de pouvoir contrôler le robot. Elle permet de faire correspondre la position de chaque moteur avec son contrôleur.

Mode Collaboratif – Programmation : après calibrage, le robot passe dans ce mode automatiquement. Il permet de programmer, éditer, charger et exécuter les programmes du robot.

Mode manuel – FreeMotion : ce mode permet de déplacer les articulations du robot à la main. Ce mode est activé soit par un appui continu sur le bouton correspondant sur le robot ou sur l'interface de programmation NiryoStudio. Ce mode sera présenté ultérieurement.

4. SÉCURITÉ

Pour cette section, il est fortement recommandé de se lire la notice d'utilisation de robot Niryo Ned2, notamment toutes les informations de sécurité. La notice est disponible via le lien suivant : https://docs.niryo.com/robots/ned2/safety-instructions/

Les points suivants répertorient seulement les informations essentielles de sécurité. Un boitier de sécurité est fourni dans le pack du robot. C'est un bouton d'Arrêt d'Urgence (AU). ATTENTION : il doit toujours être câblé au robot.

Figure 24 : Bouton d'Arrêt d'Urgence (AU)

Mise en tension du robot : Branchez le robot sur secteur avec son câble d'alimentation et branchez-y le bouton d'AU. Vérifiez que ce dernier soit désactivé. Avant d'appuyer sur le bouton de mise en Marche/Arrêt, vérifiez que le robot soit fixé correctement et stable. Mais également qu'il n'y a rien autour de lui car, lors de sa mise en marche, la calibration se fait automatiquement et le robot va bouger.

Figure 25 : Avertissement d'installation

Figure 26 : Avertissement d'installation, de mise en marche/arrêt

Figure 27 : Mise en garde

Risques liés à la tension du robot :

Un danger de haute tension est associé au robot dans :

  • L'alimentation des moteurs (12V, 11.4V, 5V)
  • Les grippers de Ned2 sont alimentés par 5V. Veillez à installer les grippers lorsque le robot est hors-tension.
Actions de sécurité :
  • Extinction d'incendie : Utilisez un extincteur au DIOXYDE DE CARBONE (CO2) en cas d'incendie.
  • Arrêt manuel du bras : si nécessaire, le robot peut être arrêté manuellement par son faible poids, sa force et sa puissance de transmission limitées. Mais il est fortement recommandé d'utiliser les fonctions d'arrêt normales (par le logiciel ou le bouton en haut de la base).

5. INDICATIONS DE L'ANNEAU LED DE LA BASE

Lexique des motifs lumineux :
  • None : les LED sont éteintes
  • Breath : l'intensité lumineuse augmente et diminue périodiquement, rappelant une respiration
  • Snake : le chenillard fait le tour des LED à vitesse modérée. Plusieurs LED se suivent
  • Chase : une LED sur trois est allumée. Toutes se décalent une LED après l'autre
  • Solide : l'ensemble de l'anneau LED est réglé sur la même couleur à la fois
  • Clignotant : Clignote une couleur selon une fréquence

Utilisations prévues

Le robot Ned2 est destiné à des buts éducatifs et de recherche et développement, principalement :

  • Apprentissage de la robotique
  • Recherche et développement en robotique
  • Simulation et prototypage de scénarios industriels complets type « Pick & Place » ou « Pick & Pack »
  • Application matérielle : connecter et contrôler le robot avec des capteurs et des actionneurs externes afin de simuler une application, tout en respectant les conditions définies dans cette instruction

Utilisations non-autorisées

  • L'utilisation comme jouet pour enfant
  • L'utilisation pour un usage industriel
  • L'utilisation pour la manipulation d'objets inappropriés (avec pinces inadaptées, objets dont la masse est supérieure à la charge maximale autorisée)
  • L'utilisation en dehors des spécifications annoncées
